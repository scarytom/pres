A Bridge Over Troubled Wat-er

Lets take a old Java Interface:

public interface Function<F, T> {
  T apply(F input);
}

and extend it:

public interface Transformer
       extends Function<Lorry, Robot>
{
  Robot apply(Lorry input);
}

Now lets mock it:

Transformer megatron = context.mock(Transformer.class);
context.checking(new Expectations() {{
  allowing(megatron).apply(myLorry);
}});

and invoke:

megatron.apply(myLorry);
